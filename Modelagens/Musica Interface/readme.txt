Modelagem de um sistema de musica onde o usuario pode: 

- O usuário pode criar playlists e adicionar músicas à elas.
- As músicas são listadas por álbum.
- Os álbuns são listados por artista.
- Uma música pode ser tocada por mais de um artista.
- Os usuários podem seguir playlists e artistas.
- Os usuários podem curtir músicas.
- As músicas de uma artista são ordenadas pela quantidade de visualizações.
- As playlists criadas por um usuário podem ser públicas ou privadas
